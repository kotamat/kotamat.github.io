“サーバレスフレームワーク jaws の導入”

JAWSって何？

  • AWSのサーバレス機能 Lambda, API Gateway, S3等に対応したフレームワーク
  • プロジェクト化することによって各Lambda functionを一つのプロジェクトで管理できる
  • デプロイもできちゃう

どうやってインストールするの？

IAMでAdminユーザを作成
